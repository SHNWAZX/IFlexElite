# Deployment Checklist

Use this checklist before deploying IFlexElite.

## Before deploy

- Run the production build.
- Confirm required environment variables are documented.
- Test the main route and important interactive screens.
- Check mobile and desktop layout at common breakpoints.

## After deploy

- Open the deployed URL in a private browser session.
- Verify key assets load without console errors.
- Record the deployment URL in the release notes or changelog.
